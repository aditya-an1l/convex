# LazyGit-like TUI

This is a basic Textual-based TUI inspired by LazyGit.

## Structure

- `app.py` - main entry point
- `git_commands.py` - functions to run git commands
- `ui/` - panels and custom widgets
- `css/` - styling for the TUI
- `data/` - optional temp files

## Usage

```bash
python app.py
